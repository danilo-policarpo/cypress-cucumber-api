# Requirements

The minimum requirements are:

* node 
* Cypress

# Setup

```
npm install
```

# Test Execution web

To run the tests in headless mode, go to main directory and execute:

```
npm run execution
```

To open the Cypress IDE, run:

```
npm run open
```
# Test Execution api

```
npm run open
```

To open the Cypress IDE, run:

```
npm run test:api
```
